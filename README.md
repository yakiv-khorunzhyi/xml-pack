# Xml
This package provides 2 classes. One for reading xml files, the second for writing.

## Requirements
| master | 1.0 |
|:----------|:---------------------:|
| extension | xmlreader & xmlwriter |
| php       | \>\=5.6               |

## Installation
Install the latest version with:
```
$ composer require yakiv-khorunzhyi/xml-package
```
